# Зависимости

### Основные зависимости для микросервисного проекта

1. **Developer Tools**:
   - GraalVM Native Support
   - Spring Boot DevTools
   - Lombok
   - Spring Configuration Processor
   - Docker Compose Support

2. **Web**:
   - Spring Reactive Web
   - Spring for GraphQL
   - Spring HATEOAS

3. **Security**:
   - Spring Security
   - OAuth2 Client

4. **SQL**:
   - Spring Data JPA
   - Flyway Migration
   - PostgreSQL Driver

5. **NoSQL**:
   - Spring Data MongoDB

6. **Сообщения**:
   - Spring for Apache Kafka
   - Spring Integration

7. **Тестирование**:
   - Spring REST Docs
   - Testcontainers

8. **Наблюдаемость**:
   - Spring Boot Actuator
   - Prometheus

9. **Шаблоны**:
   - Thymeleaf
